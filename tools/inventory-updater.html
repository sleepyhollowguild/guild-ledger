Guild Ledger

Inventory Updater

Use this whenever you export from ManaBox. It creates a fresh data/inventory.js file for the Guild Ledger without editing the app HTML. Replace only that file in GitHub.

1. ManaBox Export

Drop/select your full ManaBox CSV here.

Required columns: Name, Set code, Set name, Collector number, Foil, Rarity, Quantity, ManaBox ID, Scryfall ID, Purchase price, Condition, Language. The tool also fetches color and card type from Scryfall.

2. Showcase Prices

Optional: upload your showcase pricing CSV with a Sale Price column.

Cards over $1 need a Sale Price to become customer-checkout-ready. Unpriced showcase cards can be exported for review.
