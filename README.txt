Sleepy Hollow Guild Ledger

Open index.html in Safari/Chrome. This is a static kiosk app using your ManaBox data.

Features:
- Branded parchment/guild design
- Search, filters, sort by A-Z, price high-low, price low-high, quantity
- Trader's Pack cart and checkout share sheet
- Pull list sorted by Showcase, $1 Binder, then Bulk
- Staff report hidden behind the ⋯ button or long-pressing the SH seal
- End-of-day sold cards CSV export
- Four unavailable deck cards removed from customer search

For iPad-to-iPhone pull lists: Checkout opens the native Share Sheet. Use Messages, AirDrop, Notes, etc. A true automatic push requires a backend or Apple Shortcut webhook.


V7 updates:

- Scryfall images now request higher-resolution normal/large images instead of small thumbnails.
